# 多线程

## 多线程的实现

1. 继承Thread类

2. 实现Runnable接口

3. 采用线程池


## 多线程的优势

1. 多线程可以**减少响应时间**。单线程的话，如果有特别耗时的操作，后面的操作将都不能进行响应，必须排队等着

2. 线程的创建和切换开销小。因为他们共享进程的代码段、数据段，而创建进程将分配独立的地址空间和数据结构来维护

3. 多CPU和多核计算机本来就有执行多线程的能力，使用单线程就无法充分利用计算机资源

4. 使用多线程可以简化程序结构
